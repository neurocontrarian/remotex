# Welcome to RemoteX

**RemoteX is a visual command launcher for Linux.** Build a grid of clickable buttons — each one runs a shell command, locally or on a remote server over SSH.

![RemoteX main window](assets/main-window.png)

---

## Who is RemoteX for?

**If you're new to Linux**, RemoteX saves you from memorizing commands. Click a button to check your disk usage, update your system, or reboot — no terminal needed. RemoteX comes with 34 ready-to-use buttons out of the box.

**If you manage servers**, RemoteX is a visual SSH command launcher. Assign buttons to one or more remote machines, run commands across your infrastructure with a single click, and see the output instantly.
