# Categories

Categories group your buttons into tabs displayed just below the header bar.

![Category bar](assets/main-window.png)

## Assigning a category

When creating or editing a button, type a category name in the **Category** field. Buttons with the same category name are grouped together automatically.

You can also reassign a button quickly: **right-click → Move to category**.

## Filtering by category

Click any category pill in the tab bar to show only the buttons in that category. Click **All** to return to the full grid.

## Hiding a category

![Category right-click menu — Hide category](assets/categories-context-menu.png)

Right-click a category pill → **Hide category**. The category and all its buttons disappear from the grid.

To show it again: **Preferences → Categories → toggle the category back on**.

!!! note
    Hiding a category does not delete any buttons — they are simply filtered out of view.

## Removing a category

Categories are created and removed automatically. To remove a category, reassign or delete all buttons that belong to it — the tab disappears on its own.
