# Multi-select

!!! tip "Pro feature"
    Multi-select requires [RemoteX Pro](pro.md).

Multi-select lets you perform bulk operations on several buttons at once.

![Multi-select mode with 5 buttons selected and action bar](assets/multiselect.png)

## Entering select mode

Click the **select icon** (☑) in the header bar. The grid switches to selection mode — clicking a button toggles its selection instead of running the command.

Press `Escape` or click the select icon again to exit.

## Selecting buttons

**Click** any button to toggle its selection (highlighted in blue).

**Rubber-band** — click and drag on an empty area of the grid to draw a selection rectangle. All buttons the rectangle overlaps are added to the selection.

## Group actions

Once buttons are selected, an action bar appears at the bottom of the window:

| Action | Description |
|--------|-------------|
| **Delete** | Permanently remove all selected buttons (confirmation required) |
| **Category** | Assign all selected buttons to a category (type a new name or pick an existing one) |
| **Machine** | Assign all selected buttons to an SSH machine |
