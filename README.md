# RemoteX

A visual command launcher for Linux — a Stream Deck-style button grid for running commands without opening a terminal.

## Status

This repository is being restructured for the public release of the **free, open-source edition** of RemoteX.

The free edition will provide local command execution with a customizable button grid. Source code and binaries will be published here when ready.
